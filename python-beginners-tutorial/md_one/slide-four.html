Module One: Setting up Python

Explanation

The 

print
function, as you might guess, prints what you give to it to the screen. For example,

print('asdfhjtgryweg')

prints "asdfhjtgryweg", and

print('12345')

prints "12345". You can print numbers too:

print(4)

prints "4". Now, you might be thinking, what are those quotes for?

The quotes make Python see that what is in quotes is to be taken literally. Try these:

print('4 + 5')
print(4 + 5)

Question: what do they print? (Actually try it out first.)


Don't worry about the + in the code for now. This will be explained in Module Two.
