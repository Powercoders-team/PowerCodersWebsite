C++ for Intermediate Programmers | Module One: Basics of C++

You should complete the Python for Beginners tutorial before attempting this tutorial.

C++ is very tricky! Don't expect yourself to work things out first try!

Let's start with a simple "Hello World":

#include <iostream>
using namespace std;
int main(){
    cout << "Hello World!" << endl;
    return 0;
}

As you might see, this is several orders of magnitude more complex than the Python example.

The #include <iostream> bit tells the preprocessor to include the <iostream> header.

This allows you to use all of the basic input-output functions.

The using namespace std; bit tells the compiler to bring in all the things in the std namespace into the global namespace. Don't worry if you don't know what this means. It just means that you don't have to type

std::

if you want to access some functions.

In C++, there must be a function called main. What purpose might it serve? Can you rename it?
