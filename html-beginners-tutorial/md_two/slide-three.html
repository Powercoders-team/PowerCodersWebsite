HTML for Beginners | Module Two: Adding detail to your website

To add more color, we can add a CSS file for styling.

Create a file called main.css- this is where you can style your content.

First, let's give your heading a color.

To let the computer know we want to style the h1 element, we start it with a h1 tag with a {}.

To set the color as green, we have to set a color property followed by a : and the chosen color (red) with a ;.

main.css
h1 {
color: red;
}
