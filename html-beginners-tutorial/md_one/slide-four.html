HTML for Beginners | Module One: Basics of HTML

Time to create your first webpage!

Create a new file in your code editor. Name it index.html- it'll come in handy later.

Inside the blank workspace, give your webpage a big level one heading with the <h1> element. This will be your title.

You can name it anything you like, but for now let's stick with My Website Title to make things clear.

Your website should now look like this.

index.html
<h1>My Website Title</h1>
